# STAND UP APP

## How to run
1. ```npm install```
2. ```npm run electron```
## How to build
```npm run build```


-----
## About
This app is for people who tend to sit for a very long time in front of computers. The app is a tool for reminding users to stand up at least once every 30 mins by using a 25 + 5 clock.
